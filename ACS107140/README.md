1. 上網搜尋Virture Box 並且下載。 
![](https://i.imgur.com/lyT0vf0.png)
2. 到CentOS7下載ISO檔，一個人要求選擇不同檔案。
![](https://i.imgur.com/GeR5MlY.jpg)
3. 打開Vireture Box 新增一個虛擬電腦，選擇linux、ubuntu(64-bit)。
![](https://i.imgur.com/WZIkO0I.jpg)
##### 補充: 如果你的linux沒有64-bit的選項，在進入主機板bios設定畫面將 intel virtualization technology (vt-x) 或 AMD Virtualization(amd-v)  功能設定為enable 後重新進入系即可。
參考網址: http://www.itmaster.tw/?p=13186
![](https://i.imgur.com/CGdZypW.png)
![](https://i.imgur.com/uBKpaue.png)
4. 虛擬電腦創立好，點開設定值，點開存放裝置，在控制器裡安裝你所下載的光碟(CentOS7 ios檔)。
![](https://i.imgur.com/7xnHE3M.jpg)
5. 接下來要設定網路，啟用第一張及第二張網路卡(第二張網路卡要附加到僅限主機的網路卡，)
![](https://i.imgur.com/OXmUynk.jpg)
![](https://i.imgur.com/SA38NNf.jpg)
6. 接下來就可以啟動你的虛擬電腦。
![](https://i.imgur.com/9QD5Ak7.jpg)
![](https://i.imgur.com/NXUgS3M.jpg)
7. 點選系統中的安裝目的地來分割你的磁碟。
![](https://i.imgur.com/Bu0n1Qf.jpg)
![](https://i.imgur.com/yDt0Axb.jpg)
![](https://i.imgur.com/nRPfRQA.jpg)
8. 設定你的root以及一般用戶的帳號密碼。
![](https://i.imgur.com/wT6Y4x0.jpg)
![](https://i.imgur.com/V6OefUr.png)
![](https://i.imgur.com/Va4b6xT.png)
9. 安裝完成後就可以登入帳號開始使用你的LINUX。
![](https://i.imgur.com/KCT2iQU.jpg)
![](https://i.imgur.com/rFG58nB.jpg)
![](https://i.imgur.com/yJnvIGx.jpg)
